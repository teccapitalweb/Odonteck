# OdonTeck Consulting Group - Sitio Web

## Estructura del proyecto

```
odonteck-site/
├── index.html              ← Página principal
├── styles.css              ← Estilos globales
├── script.js               ← Lógica JavaScript
├── data.js                 ← Datos de cursos, alumnos y reseñas
├── README.md
├── assets/
│   └── img/
│       ├── logo/           ← Logo de OdonTeck
│       ├── banner/         ← Imágenes del hero
│       ├── cursos/         ← Imágenes de los 17 cursos
│       ├── alumnos/        ← Fotos de alumnos (galería)
│       └── resenas/        ← Capturas de reseñas
└── pages/
    ├── servicios.html
    ├── cursos.html
    ├── nosotros.html
    ├── galeria.html
    └── contacto.html
```

## Cómo publicar en GitHub Pages

1. Sube todos los archivos a un repositorio en GitHub
2. Ve a **Settings → Pages**
3. En **Source**, selecciona la rama `main` y carpeta `/root`
4. Guarda y espera unos minutos
5. Tu sitio estará disponible en: `https://tu-usuario.github.io/nombre-repo/`
